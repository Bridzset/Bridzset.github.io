How can you track whether an HTML mail has been opened?

The best way is to use Google Analytics Event Tracking function. Just embedding an image tag within the HTML of the email. The URL of the image contains all the information necessary for Google Analytics to understand. ( E.g campaign name, recipent ID...etc)

How can you track whether a text-only mail has been opened?

Answer:

The only way to track text-only mails is to track the clicks on the links inside the plain text emails, and they then count those as "opens", but this method is not perfect. The disadvantage here: if a plain text email is opened by a recipient, but he/she does not click on any of the links in that email then that will not be counted as an open.

What is the difference between Page Views, Page Visits and Unique Visitors of a web page? How would you count them in order to get a reliable result?

Answer:

  • Page Views: counts every page loading (if the same user refresh the page, or browsing the site, every page he/she has been visited count as one pageview every time he/she load one)
  • Page Visits: counts every user visit from different IP (including repeat visitors)
  • Unique Visitors: counts only every new visitor (or repeat visitors with expired cookie)

Order: Page Views > Page Visits > Unique Visitors

 

What is the difference between a HTTP 301 and a 302 redirect?

Answer:

  • Status 301 means that the page is moved permanently to a new location. The client/browser should not attempt to request the original location but use the new location anymore.
  • Status 302 means that the resource is temporarily located somewhere else, and the client/browser should continue requesting the original url
